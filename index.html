git config


git config –global user.name [name]
git config –global user.email [email address]

This command sets the author name and email address respectively to be used with your commits.



git init


git init

Create a new Git repository in the current directory.



git clone


git clone [url]

This command is used to obtain a repository from an existing URL.



git status


git status

This command lists all the files that have to be committed.



git add


git add [file]

This command adds a file to the staging area.


git add *

This command adds(stages) all new modifications, deletions, creations to the git.



git commit


git commit -m "message"

This command records or snapshots the file permanently in the version history.


git commit -am "message"

This command adds the file in staging area and also commits changes ( work only for modified files) .



git log


git log

This command display the version history of the current branch.



git diff


git diff

This command Display the difference between the Index and the Workspace.


git diff --staged

This command shows the differences between the files in the staging area and the latest version present in the last commit.



git reset


git reset [file]

This command unstages the file, but it preserves the file contents.


git reset --hard

Throws away all your uncommitted changes, hard reset files to HEAD



git rm


git rm [file]

This command deletes the file from your working directory and stages the deletion.



git branch


git branch

This command lists all the local branches in the current repository.

git branch [branch name]

This command creates a new branch.


git branch -d [branch name]

This command deletes the specified branch.



git checkout


git checkout [branch name]

This command is used to switch from one branch to another.


git checkout -b [branch name]

This command creates a new branch and also switches to it.



git merge


git merge [branch name]

This command merges the specified branch to current branch.

git remote


git remote -v

This command shows the remote for pull and push


git remote add [variable name] [Remote Server Link]

This command is used to connect your local repository to the remote server.


git remote

This command shows the remotes.



git push


git push [variable name] master

This command sends the committed changes of master branch to your remote repository.


git push –all [variable name]

This command pushes all branches to your remote repository.



git pull


git pull [Repository Link]

This command fetches and merges changes on the remote server to your working directory.
